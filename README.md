## 📦 Installation

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/AnnabelMate/Personal-Portfolio.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Personal-Portfolio
   ```

3. Open `index.html` in your browser, or if it's a framework-based project, install dependencies and run the development server:
   ```bash
   npm install
   npm start
   ```
